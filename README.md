# Sealo (씰로) 🦭

> 하루 전체(기상~취침)를 **물범**이 알림으로 챙겨주고, 루틴을 완료하면 **도장을 쾅** 찍어주는 루틴 앱
> Sealo = Seal(물범) + Seal(도장) — "Seal of Approval"

## 목표
- **세상에 도움이 되는** 서비스
- **트렌디하고 귀여운** 디자인 (참고: 고양이 책집사 독서 앱 스타일 — 흑백 라인 일러스트 + 포인트 컬러)
- **군더더기 없는 기능** — 직관적이고 딱 필요한 것만
- **풀스택 개발** — 커리어(대기업 이직) 목적, 모바일 앱 출시 + 웹 지원

## 실행 방법
```bash
# 백엔드 (요구사항: Docker)
cd backend
docker compose up -d      # PostgreSQL 16
./gradlew bootRun         # 첫 실행 시 JDK 21 자동 다운로드

# 프론트
cd frontend
npm install
npx expo start            # w=웹, 폰에서는 Expo Go로 QR 스캔
```
