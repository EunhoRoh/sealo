## 왜 봐줄 만한가 (고민의 흔적)

기술 나열 대신 **트레이드오프가 있었던 결정**들이 이 저장소의 본체입니다. 전체 27건은 [결정로그](docs/03-결정로그.md), 하이라이트 6건은 [포트폴리오 원페이지](https://claude.ai/code/artifact/0ec2396d-f40d-4851-97b8-914966017347)에:

- **FCM 대신 로컬 알림** — 기기가 아는 시간을 서버가 푸시할 이유가 없다. 인프라 3종 제거 (결정로그 #16)
- **N+1을 구조로 차단** — 전 관계 LAZY + DTO 프로젝션, 핵심 화면 쿼리 2회, OSIV off ([개발원칙](docs/07-개발원칙.md))
- **AI 기능을 LLM 없이 선출시** — 생성기 인터페이스 + 룰 엔진 6테마, LLM은 구현체 교체로 (결정로그 #26)
- **실패 과금은 재화로만** — 유사투자자문업 규제로 첫 아이디어를 기각한 경험이 원칙이 됨 (결정로그 #4, #11)
- **트러블슈팅 7건 기록** — Boot 4 Jackson 3 마이그레이션, 벌크 UPDATE와 dirty checking, CORS 실구동 발견 등 ([docs/10 C절](docs/10-포트폴리오.md))

## 실행 방법

```bash
# 백엔드 (요구사항: Docker, JDK는 Gradle이 자동 프로비저닝)
cd backend
docker compose up -d      # PostgreSQL 16
./gradlew bootRun

# 프론트
cd frontend
npm install
npx expo start            # w=웹 브라우저, 폰은 Expo Go 앱으로 QR 스캔
```

상세(문제 해결 포함): [docs/11-시스템구성-배포.md](docs/11-시스템구성-배포.md)
