# 13. UX 리뷰 — 솔직한 자체 검토 (듀오링고 렌즈)

작성일: 2026-07-08
방법: 듀오링고 리텐션 메커니즘과 1:1 대조 + 사용자 여정(첫 실행→7일차) 시뮬레이션

## 듀오링고 리텐션 공식 대조표

| 듀오링고 장치 | Sealo 현황 | 판정 |
|---|---|---|
| 스트릭 (연속 기록) | 🔥 스트릭 + 칭호 6단계 | ✅ 있음 |
| **손실 회피** (스트릭 끊김 공포) | 저녁 리마인드 없음 — 잊으면 그냥 끊김 | ❌ **오늘 수정** |
| 첫 세션에 바로 성공 경험 | 온보딩 후 빈 홈 (루틴 0개, 직접 만들어야) | ❌ **오늘 수정** |
| 보상 즉시 체감 (보석 잔액 상시 노출) | 조개 잔액이 상점에만 보임 | ❌ **오늘 수정** |
| 캐릭터 감정 (Duo의 죄책감 유발) | 물범 멘트/시무룩 스누즈 | ✅ 방향 맞음, 더 강화 여지 |
| 마이크로 인터랙션 (효과음/애니) | 도장 쾅 Lottie + 햅틱 | ✅ 있음 |
| 일일 목표 설정 | 없음 | 📋 백로그 |
| 리그/친구 (사회적 압력) | 없음 (M5 응원 도장 계획) | 📋 백로그 |

## 사용성 구멍 (기능 완결성)

| 문제 | 심각도 | 조치 |
|---|---|---|
| **루틴 삭제/수정 UI가 없음** (API는 있는데 화면에 없음 — 오타 루틴을 지울 방법이 없다!) | 🔴 치명 | 삭제 **오늘 수정**, 수정은 P1 |
| 요일 선택 없음 (전부 매일 고정) | 🟠 높음 | P1 |
| 시간 입력이 "HH:MM" 텍스트 타이핑 | 🟠 높음 | P1 (크로스플랫폼 타임피커) |
| 스트릭 최고 기록(최장) 미표시 | 🟡 | P2 |
| 다크모드 없음 | 🟡 | P2 (토큰 구조라 준비됨) |
| 플랜 목표일도 텍스트 입력 | 🟡 | P1 타임피커와 함께 |

## 오늘 수정한 것 (P0 퀵윈 4개)

1. **물범 추천 루틴으로 시작** — 빈 홈에서 원탭으로 시작 루틴 3개 생성 (기상 07:00 따르릉 / 물 한 잔 09:00 / 취침 준비 23:00 따르릉) → 첫 세션에 바로 도장 찍을 거리를 준다
2. **저녁 스트릭 지킴 알림** — 매일 21:30 "오늘 도장 다 찍었어? 스트릭 지키자 🔥" (손실 회피 장치)
3. **홈에 조개 잔액 상시 노출** — 보상 체감
4. **루틴 삭제** — 길게 누르면 확인 후 삭제

## 다음 개선 로드맵

- **P1 ✅ 완료 (2026-07-08)**: 루틴 수정/삭제 통합 모달(길게 누르기), 요일 선택 칩(최소 1개 보장), 네이티브 타임피커(iOS compact/Android 다이얼로그, 웹은 텍스트 폴백), 플랜 날짜 피커
- **P2**: 일일 목표(오늘 N개 목표 링), 최장 스트릭, 다크모드, 도장 찍을 때 효과음
- **P3 (M5 잔여와 연계)**: 홈 위젯, 친구 응원 도장(사회적 압력), 스트릭 복구권(수익화 연결)

## 종합 판정

핵심 루프(알람→도장→보상→꾸미기)와 감정 장치(물범/도장/칭호)는 듀오링고 문법을 잘 따르고 있음.
**약점은 "쓰다가 마는 부분"** — 첫 실행의 빈 화면, 잊었을 때 잡아주는 손, 기본 CRUD 완결성.
오늘 P0를 막았고, P1(입력 편의)까지 끝나면 지인 테스트(TestFlight/내부 테스트) 돌릴 만한 상태.
