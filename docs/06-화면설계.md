# 06. MVP 화면 설계 (와이어프레임)

작성일: 2026-07-07
디자인 언어: 흑백 라인 드로잉 물범 + 화이트 배경 + 포인트 컬러 1개 (책집사 앱 벤치마킹).
포인트 컬러 후보: 바다 느낌의 **딥 블루** 또는 도장 인주 느낌의 **레드** — 도장 컨셉과 직결되는 레드 추천.

## 화면 목록 (MVP)

| # | 화면 | 탭 |
|---|---|---|
| 0 | 온보딩 (3장) + 소셜 로그인 | - |
| 1 | 홈 — 물범 + 오늘의 루틴 | 탭1 |
| 2 | 루틴 등록/수정 (모달) | - |
| 3 | 상점 — 꾸미기 | 탭2 |
| 4 | 기록 — 캘린더 도장판 + 스트릭 | 탭3 |
| 5 | 계정/설정 | 탭4 |
| 6 | 푸시 알림 (시스템 UI) | - |

## 0. 온보딩 (첫 실행, 3장 스와이프)

책집사 앱의 감성 카피 벤치마킹. 마지막 장에서 로그인.

```
[1장] 물범 일러스트 (뒹굴거리는 모습)
      "나는 물범. 하루 종일 뒹굴거리는 게 특기야"
[2장] 물범이 도장을 들고 있는 모습
      "네가 루틴을 지킬 때마다 내가 도장을 쾅 찍어줄게"
[3장] "준비 됐지? 오늘부터 같이 해보자 🦭"
      [ 카카오로 시작하기 ]  [ Google로 시작하기 ]
```

- 로그인 직후: 첫 루틴 1개를 바로 만들게 유도 ("먼저 아침 루틴 하나만 정해볼까?") — 빈 홈 화면 방지

## 1. 홈 — 물범 + 오늘의 루틴 (핵심 화면)

```
┌─────────────────────────────┐
│  Sealo            🐚 120    │  ← 재화(조개) 잔액
│                             │
│        (물범 일러스트)        │  ← 꾸미기 아이템 착용 상태로 렌더
│   ┌─────────────────────┐   │
│   │ 오늘 3개 남았어! 화이팅 │   │  ← 물범 말풍선 (상황별 멘트)
│   └─────────────────────┘   │
│                             │
│  🔥 연속 5일째               │  ← 스트릭
│                             │
│  오늘의 루틴                  │
│  ┌─────────────────────┐    │
│  │ ✅ 07:00 기상        │    │  ← 완료: 작은 도장 찍힘
│  │ ✅ 07:10 세수        │    │
│  │ ⬜ 12:30 점심 산책    │    │  ← 탭하면 도장 쾅 애니메이션
│  │ ⬜ 22:00 독서        │    │
│  │ ⬜ 23:30 취침        │    │
│  └─────────────────────┘    │
│                    [+ 추가]  │
│  🏠홈  🛍상점  📅기록  👤계정  │
└─────────────────────────────┘
```

- **루틴 체크 = 이 앱의 가장 중요한 순간.** 물범이 도장 들고 화면 중앙에 쾅 찍는 풀스크린 애니메이션 (1초 내외, 스킵 가능) + 조개 +N
- 하루 전체 완료 시: 큰 Seal of Approval 연출 + 물범 축하 멘트
- 말풍선 멘트: 시간대/달성률 따라 변화 (아침 인사, 미완료 리마인드, 전체 완료 축하)

## 2. 루틴 등록/수정 (바텀 시트 모달)

```
┌─────────────────────────────┐
│  루틴 만들기                  │
│  이름   [ 점심 산책        ]  │
│  아이콘 [ 🚶 선택 ]           │
│  시간   [ 12:30 ]            │
│  반복   [월][화][수][목][금][토][일] │
│  알림   [ ON ⭘ ]            │
│         [ 저장 ]             │
└─────────────────────────────┘
```

- 필드 5개가 전부. "군더더기 없이" 원칙 — 메모, 태그, 색상 같은 건 넣지 않는다

## 3. 상점 — 꾸미기

```
┌─────────────────────────────┐
│  상점                🐚 120  │
│  (물범 미리보기 — 착용 시뮬)   │
│  [전체] [모자] [안경] [목도리] │
│  ┌────┐ ┌────┐ ┌────┐      │
│  │밀짚 │ │선원 │ │도토리│      │
│  │모자 │ │모자 │ │모자 │      │
│  │🐚80│ │🐚100│ │🐚60 │      │
│  └────┘ └────┘ └────┘      │
│  🏠홈  🛍상점  📅기록  👤계정  │
└─────────────────────────────┘
```

- MVP: 카테고리 2~3개, 아이템 6~10종이면 충분 (책집사도 모자 몇 종으로 시작하는 구조)
- 보유 아이템은 "착용" 토글. 현금 결제 없음 (Phase 2에서 프리미엄 아이템 추가)

## 4. 기록 — 캘린더 도장판 + 스트릭

```
┌─────────────────────────────┐
│  기록                        │
│  🔥 연속 5일  |  🏆 최고 12일  │
│  ◀    26년 7월    ▶         │
│  일 월 화 수 목 금 토           │
│         1  2  3  4          │
│  5  6  🦭 🦭 🦭 10 11        │  ← 전체 달성일 = 물범 도장
│  ...                        │
│  이번 주 달성률  ▓▓▓▓░ 80%    │
│  🏠홈  🛍상점  📅기록  👤계정  │
└─────────────────────────────┘
```

- 부분 달성일은 흐린 도장, 전체 달성일은 선명한 큰 도장 — 도장판 채우는 수집 욕구
- 통계는 주간 달성률 바 하나만. 상세 통계는 Phase 2 (Plus 혜택 후보)

## 5. 계정/설정

- 프로필 (닉네임, 로그인 계정), 알림 설정(방해금지 시간대), 문의/약관, 로그아웃/탈퇴
- 끝. (설정이 많아지면 실패한 것)

## 6. 푸시 알림

```
🦭 Sealo — 지금 12:30
점심 산책 시간이야!
[ 🦭 도장 쾅 ]  [ 30분 뒤에 ]
```

- 액션 버튼으로 앱 안 열고 완료 처리 (결정로그·04 문서의 도장 메커니즘)
- "30분 뒤에" 스누즈 1회 제공
