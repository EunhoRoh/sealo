## 제품의 심장: 알림이 아니라 "알람" ⏰

> Ethan 요구사항 원문: "사람들의 루틴 시간이 되면 **폰 알람이 울려야** 되는거야" — 기상 알림처럼.

| | 알림 (Notification) | **알람 (Alarm)** |
|---|---|---|
| 소리 | 딩— 한 번 | **따르릉 계속 울림** |
| 무시 가능성 | 높음 (배너 스쳐감) | 낮음 (꺼야만 멈춤) |
| 현재 Sealo | ✅ 이 수준 | ❌ 이걸 만들어야 함 |

**기술 현실 (정직하게)**:
- **Android**: 진짜 알람 가능. `AlarmManager`(정확한 시각 보장) + 풀스크린 인텐트(잠금화면 위로 알람 화면) + 지속 사운드. 단 Expo 기본 모듈로는 안 되고 **네이티브 모듈**(expo-alarm 계열 or 직접 작성) 필요 → M1의 핵심 기술 과제
- **iOS**: 서드파티 앱은 시스템 알람 사용 불가 (Apple 정책). 대안: ① 30초 길이 사운드 알림 + 연속 반복 예약(알라미 등이 쓰는 기법) ② 라이브 액티비티로 잠금화면 점유. **iOS에서 100% 알람은 불가능함을 인정하고 최선의 근사치로 설계**
- 이 조사·구현 스토리 자체가 포트폴리오 A급 소재

## 마일스톤

### M0. MVP 코어 ✅ (2026-07-07 완료)
루틴 CRUD, 로컬 알림, 도장 쾅(Lottie), 조개, 상점(꾸미기 6종), 캘린더 도장판, 스트릭, 온보딩, 4탭.

### M1. "따르릉" 알람 경험 + 실기기 검증 ← **다음**
목표: 알림을 알람으로 격상. Sealo의 존재 이유를 완성.
- [ ] 실기기(Expo Go) 검증: 현 로컬 알림 동작 확인 → 결과를 docs/10에 기록
- [ ] **Expo Dev Client 전환** (네이티브 모듈 쓰기 위한 선행 작업)
- [ ] Android 알람: AlarmManager + 풀스크린 알람 화면 (물범이 화면 가득 "일어나! 세수할 시간이야!" + 도장 쾅 버튼/스누즈)
- [ ] iOS 근사 알람: 30초 사운드 + 반복 예약
- [ ] 알람음: 기본 따르릉 + **물범 울음소리 알람** (상점 판매 후보, docs/05)
- [ ] 루틴별 알람/알림 선택: 기상은 알람, 점심 산책은 가벼운 알림 — 사용자가 강도 선택
- 창의 요소 💡: **"물범이 깨우러 옴"** — 알람 화면에서 물범이 점점 다가오는 연출. 스누즈 누르면 물범이 시무룩해지고, 3번 밀면 뒹굴며 포기 (죄책감 디자인, Forest의 나무 죽음 공식)

### M2. 소셜 로그인 + 클라우드 배포
목표: 남에게 보여줄 수 있는 상태.
- [ ] 카카오/구글 로그인 + JWT (키 발급 대기, docs/09) / 게스트 데이터 → 계정 연동 이전
- [ ] Flyway 마이그레이션, 프로필 분리, Dockerfile, GitHub Actions CI/CD
- [ ] AWS 배포 (docs/11) + EAS Build로 스토어 내부 테스트 트랙
- [ ] 스토어 등록 준비: 스크린샷, 설명, 개인정보처리방침

### M3. 테마 플랜 시스템 (Ethan 원안 기능들의 본체)
목표: 루틴(반복)을 넘어 "계획"까지. **Plan 도메인 하나로 일반화** (docs/04).
- [ ] 공통 엔진: 플랜 = 이름 + 항목 리스트 + 날짜/반복 + 알림 + 진행률
- [ ] 🧴 **세안 플랜**: 제품 순서 체크리스트 (토너→세럼→크림), 아침/저녁 구분. 창의 💡: 제품 개봉일 등록하면 **유통기한 임박 알림** ("비타민C 세럼 이제 3주 남았어!")
- [ ] 💪 **운동 플랜**: 요일별 운동 구성 (월: 가슴, 수: 하체…), 세트/횟수 체크. 창의 💡: 완료 시 물범이 같이 운동한 것처럼 근육(?) 포즈
- [ ] 📚 **공부 플랜**: 하루/주/월 계획 트리, 과목 배정, **뽀모도로 타이머** (25분 집중 — 물범이 옆에서 같이 공부하는 화면, 폰을 뒤집으면 물범이 잠듦). 주간 리뷰 자동 생성
- [ ] ✈️ **여행 플랜**: D-day 카운트다운, 준비물 체크리스트(템플릿: 국내/해외/캠핑), 방문 장소 리스트, 전날 밤 "짐 쌌어?" 알림. 창의 💡: 여행 완료 시 캘린더에 **특별 여행 도장** (비행기 탄 물범)
- [ ] ➕ **커스텀 테마**: 사용자가 직접 테마 생성 + 창의 💡: 잘 만든 테마를 **템플릿으로 공유** (v4 소셜의 씨앗)
- [ ] 구글 캘린더 연동: 플랜/루틴을 캘린더에 양방향 표시
