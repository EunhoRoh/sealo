작성일: 2026-07-07
원칙: 핵심 루프는 무료, 과금·재화 소비는 감정/꾸미기/편의에만 (docs/05)

## 카테고리 로드맵

| 카테고리 (enum) | 내용 | 시점 |
|---|---|---|
| 🎩 꾸미기 (ACCESSORY) | 물범 착용 아이템. 바다 세계관: 불가사리 머리핀, 해초 목도리, 소라고둥 모자… | **MVP ✅ (구현됨)** |
| 🦭 도장 (STAMP) | 도장 디자인 교체: 물범 발바닥, 하트, 별, **"참 잘했어요"**(한국 감성) | v1.1 |
| 💬 물범의 한마디 (VOICE) | 말풍선 말투 팩: 존댓말/부산 물범("루틴 했나?")/할머니("아이고 장하다")/열정 코치. 알람 문구까지 적용 | v1.1 — **제작비 최저(텍스트), 애착 효과 최고. 우선 추천** |
| 🏠 보금자리 (THEME) | 홈 배경: 빙하, 바닷속, 물범 온천, 우주 | v1.2 |

- 메뉴 구조: 상단 카테고리 칩. 카탈로그에 존재하는 카테고리만 자동 노출 (프론트가 서버 데이터 기반으로 칩 생성 → 카테고리 추가 시 프론트 수정 불필요)
- Phase 2 구독(Sealo Plus) 전용 아이템은 카탈로그에 잠금 표시로 노출 (책집사의 "구독 전용" 배지 방식)

## MVP 구현 스펙 (2026-07-07)

- **가격 밸런스**: 도장 1개 = 조개 10개. 하루 루틴 5개면 50조개/일 → 아이템(50~120조개)은 1~3일 분량. 첫 구매까지 빠르게, 이후 점점 비싸게
- 시드 아이템 6종: 불가사리 머리핀(50) / 해초 목도리(60) / 동그란 안경(70) / 밀짚모자(80) / 선원 모자(100) / 소라고둥 모자(120)
- 같은 카테고리는 1개만 착용 (착용 시 기존 자동 해제)
- **assetKey 체계**: 서버는 `acc_straw_hat` 같은 논리 키만 저장, 실제 그림은 프론트 `seal-character.tsx`의 맵에서 결정 → 이미지 교체 시 서버 무관 (docs/07 유지보수 원칙)

## API
- `GET /api/shop/items` — 카탈로그 + 보유/착용 여부 (쿼리 2번)
- `POST /api/shop/items/{id}/purchase` — 잔액 부족/중복 구매 409
- `POST /api/shop/items/{id}/equip`, `/unequip`
- `GET /api/members/me` — 닉네임 + 조개 잔액
