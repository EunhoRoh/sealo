# 01. 아이디어 후보

작성일: 2026-07-07

## 참고 앱 (디자인 벤치마킹)
고양이 "책집사" 컨셉의 독서 기록 앱 스크린샷 참고.
- 흑백 라인 드로잉 캐릭터(고양이) + 미니멀한 UI
- 포인트 컬러(빨강 계열) 최소 사용
- 재화(발바닥 포인트)로 캐릭터 꾸미기(모자/안경/의상) + 스타일(테마 색상) 판매 → 수익화
- 구독 전용 아이템 존재
- 상단 배너 광고
- 하단 탭 5개: 홈 / 상점 / 서재 / 기록 / 계정
- 기록 탭: 캘린더 + 통계 + 결산
- 캐릭터가 말풍선으로 한마디 ("오늘도 책과 좋은 시간 보냈어?") → 정서적 애착 유도

**배울 점**: 핵심 루프(기록) + 귀여운 캐릭터 + 꾸미기 수익화 + 광고. 기능이 적어도 애착으로 리텐션을 만든다.

---

## 후보 A: 귀여운 주식 앱

### 컨셉
- 왕초보용 주식 공부 콘텐츠 (하나하나 쉽게)
- 뉴스 크롤링 / 한국투자증권 API 등으로 정보 수집
- **1,000원 결제 시 AI(LLM) 추천 종목 3개 제공**

### 장점
- 수익 모델이 명확함
- 주식 데이터 API + LLM 연동 → 기술적으로 배울 게 많음

### 단점 / 리스크 ⚠️
- **법적 리스크 (중요)**: 한국에서 대가를 받고 특정 종목을 추천하면 **유사투자자문업 신고 대상** (자본시장법). 미신고 영업은 처벌 대상. 개인 사이드 프로젝트로 유료 종목 추천은 사실상 불가.
- AI 추천 종목으로 사용자가 손실 보면 분쟁/책임 문제
- 한국투자 API는 실계좌/모의계좌 발급 등 진입 절차 필요
- LLM 종목 추천의 품질을 보장할 수 없음

### 살릴 수 있는 방향 (피벗)
- 유료 추천을 빼고 **"주식 왕초보 교육 + 모의투자 연습"** 앱으로 전환하면 법적 문제 없음
- 수익화는 광고 + 프리미엄 콘텐츠(교육 콘텐츠는 자문이 아님)

## 후보 B: 하루 루틴 알림 앱 (따르릉)

### 컨셉
- 기상 → 세수 → 출근 → 식사 → 공부 → 폰 잠금 → 취침까지 **하루 전체를 알림으로 챙겨주는** 앱
- 세부 루틴: 스킨케어 순서(나이아신아마이드, 비타민C 등), 운동 루틴, 뽀모도로 공부
- 폰 잠금 시간엔 읽을거리 제공
- 구글 캘린더 연동 가능
- 컴팩트하고 심플한 일정 뷰

### 장점
- **법적 리스크 없음**, 세상에 도움 되는 방향과 부합
- 캐릭터/꾸미기 수익화 모델(참고 앱)과 궁합이 좋음 — "루틴 지킬 때마다 재화 획득 → 캐릭터 꾸미기"
- 푸시 알림, 스케줄링, 캘린더 연동, 통계 등 **풀스택 포트폴리오로 보여줄 기술 요소가 풍부**
- MVP를 작게 시작 가능 (루틴 등록 + 알림만으로 시작)

### 단점 / 리스크
- 경쟁 앱 존재 (마이루틴, Routinery, 투두메이트 등) → **차별화 = 귀여운 캐릭터 + 극단적 심플함**
- 기능 욕심 부리면 금방 비대해짐 (스킨케어+운동+뽀모도로+읽을거리... 전부 넣으면 참고 앱의 미덕을 잃음)
- 알림의 신뢰성(안드로이드 배터리 최적화, iOS 백그라운드 제한) 구현 난이도 있음

| 법적 안전성 | ❌ 유료 추천은 규제 대상 | ✅ 문제 없음 |
| 세상에 도움 | △ | ✅ |
| 귀여운 디자인 궁합 | △ (돈 얘기 + 귀여움은 어색할 수 있음) | ✅ 캐릭터 동기부여와 찰떡 |
| 포트폴리오 가치 | ○ (외부 API, 결제) | ✅ (푸시, 스케줄링, 통계, 캘린더 연동) |
| MVP 작게 시작 | 어려움 | 쉬움 |

**추천: B (루틴 앱)** — 단, 참고 앱처럼 "핵심 루프 하나(루틴 체크) + 캐릭터 애착"으로 좁게 시작하고, 스킨케어/운동/뽀모도로는 이후 확장 기능으로.
