# 04. MVP 기능 정의 (초안 — 논의 필요)

작성일: 2026-07-07 (수익화 리서치 반영: [05-수익화전략.md](05-수익화전략.md))
원칙: **참고 앱(책집사)처럼 핵심 루프 하나만.** 나머지는 전부 v2 이후로.
수익화 원칙: MVP는 과금 없음. 단, **재화+상점 경제 구조는 MVP부터** 탑재 (나중에 붙이기 어려움). 광고→구독은 Phase 1~2 순차 도입.

## 핵심 루프
```
루틴 등록 → 시간 되면 알림 (따르릉) → 완료하면 물범이 도장 쾅 🦭 → 재화 획득 → 캐릭터 꾸미기
```

### 도장 메커니즘 (2026-07-07 확정)
- 도장은 **알람 시점이 아니라 "루틴 완료 시"** 찍는다 — 해낸 것에 대한 승인(approval)이어야 의미가 있고, 알람만으로 찍히면 보상 루프가 무너짐
- 단, **푸시 알림에 액션 버튼 [🦭 도장 쾅]** 을 달아 앱 안 열고 알림에서 바로 완료 처리 (마찰 최소화)
- 개별 루틴 완료 = 작은 도장 / **하루 전체 달성 = 캘린더에 큰 "Seal of Approval" 도장** → 기록 탭이 도장 수집판이 됨
- 확장(v2): 도장 디자인을 상점에서 판매

## MVP에 넣는 것 ✅

### 1. 루틴 (핵심)
- 루틴 등록: 이름, 시간, 반복 요일, 이모지/아이콘
- 푸시 알림 (FCM)
- 오늘의 루틴 목록 + 체크 (완료 처리)

### 2. 캐릭터 (애착) — 물범 🦭 확정
- 루틴 완료 시 물범이 도장 찍는 애니메이션 + 재화 획득
- 홈 화면에서 물범이 한마디 ("세수할 시간이야!", "오늘 루틴 다 지켰네, 대단해!")

### 3. 상점 (수익화 기반)
- 재화로 캐릭터 꾸미기 아이템 구매 (모자 몇 종 정도로 시작)

### 4. 기록 + 스트릭 (리텐션)
- 캘린더 뷰: 날짜별 도장 (하루 전체 달성 = 큰 Seal of Approval)
- **연속 달성(스트릭) 카운터** — 리서치 결과 습관 앱 리텐션·결제의 핵심 엔진 (듀오링고 스트릭, Phase 2의 "도장 복구" 결제 동기가 됨). MVP부터 필수
- 간단한 통계 (주간 달성률 정도)

### 5. 계정
- 소셜 로그인 (카카오 + 구글)

### 화면 구성 (하단 탭 4개)
| 탭 | 내용 |
|---|---|
| 홈 | 캐릭터 + 오늘의 루틴 체크리스트 |
| 상점 | 꾸미기 아이템 |
| 기록 | 캘린더 + 통계 |
| 계정 | 로그인/설정 |

## MVP에서 뺀 것 ❌ (v2 이후 확장 후보 백로그)

### 테마 플랜 시스템 (Ethan 원안 기능들의 통합 설계, 2026-07-07 여행 추가)
루틴(반복)과 별개의 **"계획(플랜)" 도메인**. 테마 템플릿을 고르면 그에 맞는 체크리스트/단계/일정 틀이 생기고, 사용자가 새 테마를 직접 만들 수도 있다:
- 🧴 **세안 플랜**: 사용할 제품(나이아신아마이드, 비타민C 등)과 바르는 순서 — 루틴의 "세부 단계"로 연결
- 💪 **운동 플랜**: 어떤 운동을 어떤 순서로, 요일별 구성 + 알림
- 📚 **공부 플랜**: 하루/일주일/한달 계획, 과목 배정, 뽀모도로 타이머 알림
- ✈️ **여행 플랜** (2026-07-07 추가): 특정 날짜에 어디 갈지, 준비물 체크리스트, 방문 장소 목록
- ➕ **커스텀 테마**: 사용자가 새 테마 플랜을 직접 생성
- 설계 포인트: 전부 "이름+항목 리스트+날짜/반복+알림"의 변형이라 **하나의 Plan 도메인으로 일반화** 가능. MVP 루틴 검증 후 착수

## 캐릭터 동물 후보 분석 (2026-07-07) — 물범으로 확정 ✅

| 최종 후보 | 강점 | 약점 |
|---|---|---|
| 🦭 물범 ✅ | 영어 seal = 도장 → **"seal of approval"**: 루틴 완료 시 물범이 도장 쾅. 글로벌 언어유희 + 마스코트 희소성. "게으른 물범을 내 루틴으로 성장시킨다" 서사 | 실제 루틴 습성은 약함 (서사로 커버) |
